# Гайдлайн розмітки

> Формат: YOLO bbox (`class_id x_center y_center width height`, нормалізовано 0-1).
> Тулза: рекомендую CVAT (self-hosted) або Roboflow для початку.

## Bounding box

- **Tight fit:** бокс має торкатись усіх країв об'єкта без зайвого «повітря» по периметру. Допустимий запас — ≤2 пікселі.
- **Цілий об'єкт:** включати весь силует разом з башточкою/стволом/антенами.
- **Тіні і пил:** НЕ включати в бокс.
- **Мінімальний розмір:** не розмічати об'єкти <16×16 px (пропустити).

## Truncation (об'єкт обрізаний кадром)

- Якщо видно ≥50% об'єкта — розмічати по видимій частині.
- Якщо <50% — пропустити (не розмічати).

## Occlusion (перекриття)

- Розмічати по гіпотетичних межах об'єкта, навіть якщо частина перекрита деревом/будівлею, **якщо** перекрито <50%.
- Якщо перекрито >50% — пропустити.

## Кілька об'єктів поруч

- Колона техніки в русі — кожна машина окремий бокс.
- Скупчення людей `people_group` — **один** бокс на всю групу (не кожен солдат окремо), якщо вони компактно.
- Якщо група розтягнута на >2× ширини найбільшої машини — розбивати на кілька боксів.

## Спалена техніка

- **НЕ розмічати** як ціль. Залишити кадр без боксів — це hard negative.
- Якщо в кадрі є і спалена, і боєздатна — розмітити тільки боєздатну.
- Критерій «спалена»: візуально вигоріла, відсутні гусениці/колеса/башта, дим/копоть.

## Сезонність і ландшафти

Збір повинен покривати:
- **Сезони:** літо (зелень), зима (сніг, гола рослинність), міжсезоння (багнюка).
- **Ландшафти:** лісосмуга (посадка), ліс, відкрите поле, ґрунтова дорога з багнюкою, місто, асфальтована дорога.

Маркувати метадані кожного кадру (JSON sidecar):

```json
{
  "image_id": "img_00123",
  "source": "synthetic|real",
  "season": "summer|winter|mud_season",
  "landscape": "forest_belt|forest|field|mud|urban|road",
  "weather": "clear|fog|rain|snow",
  "time_of_day": "day|dusk|night"
}
```

Це потрібно для stratified split і per-bucket evaluation (real vs synthetic, summer vs winter).

## Контроль якості

- **Double-pass:** перші 100 кадрів кожного нового аннотатора рев'юить senior.
- **IoU consistency check:** на 5% кадрів повторна розмітка іншим аннотатором, цільовий IoU ≥0.85.
- Помилкові розмітки (FP, FN) логуються в `dataset/qa_log.md`.

## Заборонене

- Не змінювати ID класу заднім числом без міграції всього датасету.
- Не зменшувати box щоб «не залазив» на сусідній об'єкт — boxes можуть перекриватись.
- Не вгадувати клас якщо не впевнений: ставити `unsure` тег і виносити на рев'ю.
